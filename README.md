## Churn Predict with Logistic Regression from scratch

Project to predict customer churn.

Developed a Logistic Regression model from scratch with just numpy and trained with Churn Modeling Dataset from Kaggle. [Access the dataset here](https://www.kaggle.com/datasets/shrutimechlearn/churn-modelling)

## Setup

Execute the following commands on prompt command to run the environment locally:

Create a virtual environment:

```
python -m venv .venv
```

Activate the virtual environment:
```
env\Scripts\activate.bat
```

Install dependencies
```
pip install -r requirements.txt
```
